# Way Maker Logo – Grok Imagine (Flux) Prompt

Copy the prompt below into Grok Imagine for professional logo concepts. Use as-is or adjust the last line for layout (horizontal vs stacked).

---

## Single prompt (paste this)

**Professional minimalist vector logo, transparent background, flat design, sharp edges, high contrast. Primary focus: bold typography "WAY MAKER" in heavy contemporary sans-serif (Poppins ExtraBold style), uppercase, clean letterforms, dark navy #0F172A. Below it, smaller and lighter weight: tagline "Sunshine Coast Rubbish Removal" in a complementary sans-serif, charcoal #1E293B, clearly readable. Centered, balanced composition with excellent negative space.**

**Icon/symbol: a single elegant, symbolic mark integrated above or beside the wordmark—either (1) a stylized load or tidy pile / clearance motif with a clear path or arrow suggesting "making a way," or (2) a strong directional arrow or open road motif—minimal, one or two clean shapes, no clutter. The icon should feel like "clearing a path": confident, helpful, not cartoonish. No overflowing trash, no messy details; think premium service brand, modern Australian eco-company.**

**Color palette, strict: emerald green #10B981 as primary accent (use in icon or key letter); vibrant orange #F97316 as secondary accent (one small pop—e.g. dot, path line, or highlight); dark navy #0F172A for main text; supporting charcoal #1E293B for tagline; background transparent or pure white #FFFFFF. Optional tiny accent: sunny yellow #FBBF24 on one small element for Queensland coastal feel. No gradients except an extremely subtle one if needed; flat fills preferred.**

**Style: scalable vector-style logo, perfect for website headers, truck signage, uniforms, business cards, and social media. Professional Sunshine Coast junk removal brand—trustworthy, energetic, eco-conscious, Queensland sunshine and coastal vibe. Clean lines, symmetry where appropriate, no realistic photos, no cartoon explosions or cheesy trash imagery. Inspiration: bold modern service logos, Australian eco-brands, minimal truck or clearance icons with strong typography. --ar 3:2**

---

## Layout variants (change the last part of the prompt)

- **Horizontal (headers, truck, signage):** Keep prompt as above; end with `--ar 3:2` or `--ar 16:9`.
- **Stacked/vertical (social, favicon, square):** Replace the last line with: `Stacked layout: "WAY MAKER" on top, tagline below, icon above both; square format. --ar 1:1`
- **Monochrome version:** Add at the start: `Monochrome logo, black and white only, no colour.` and remove the color palette paragraph (or replace with: "Strictly black #000000 and white only, transparent background.").

---

## Tips for Grok Imagine

1. Run the **full colour** prompt first; then ask for "same logo, monochrome black and white" if the tool allows follow-up.
2. For horizontal vs stacked, run twice with the different `--ar` and layout lines.
3. If the icon feels too busy, add: "Icon must be a single simple shape, maximum two elements."
4. To stress typography: "Wordmark is 70% of the design; icon is small and subtle."
